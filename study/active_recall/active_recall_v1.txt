sit down read for 5 times but disappear after few hours.
the brain is designed to forget almost everything. at the survival perspective, remembering everything constantly will destroy focus. if its not important than disappear.
this is not about intelligent or how good is our brain. but this is how the brain works. the brain mechanics.
rereading, highlighting, endless lecture videos, etc works because passive input doesn't convince the brain its something matter. it makes the words look familiar on the 2nd or 3rd pass, the brain mistakes recognition for actual knowledge. but mean nothing to our brain.
neuroscience studies show that the brain remember faster when struggle correctly, not harder. if understand this than, memorize become easier.

below are the top student uses to make brain remember faster, retain longer and recall memory under pressure - actual memory mechanics:

1. stop trying to store information. memory does not work like saving file to computer.
most people keep information like saving files, read, repeat, read, repeat.
the memory build through reconstruction, brain straighten the information when try to retrieve not when receive it. this called active recall.
the tiny struggle build the memory pathway. the harder retrieval feel, the stronger the memory become afterwards.
this is how the top performer do, read page, close book, force the brain to retrieve/recall. difficulty during recall = the brain is adapting.
when start reading/learning, remember the question = what i can recall without looking? 

2. brain love incomplete patterns. pattern-detection machine. when something incomplete, then it create a kind of mental tension that the brain wants to resolve.
remember the old memories but not yesterday memories, because brain love the patterns not isolated facts.
random information disappear quickly. connected information survive.
turn the information into relationship instead of raw data. turn the information into stories, diagrams, pictures works well. 
meaning create memory hook, more memory hook, fast to recall.
memorize faster = how do i to connect this better?

3. the 24 hours rule.
forgetting is fast, within 24 hours. if no revisited the memory then the brain labeled the memory as temporary.
memory require space reinforcement. space = memory strengthen when learning is spread ou over time.
study once every 3 hours - weak retention.
study 1 hour over 3 days - much stronger memory.
reinforcement (repetition + recall) - memory needs revisited and used.
spaced repetition = craved into stone slowly.
so always revisited the memory not too early or not too late.
day1 learn, day2 review, day4 review, day7 review, day14 review.
cramming works but disappear faster.

4. emotion is memory superpower.
brain prioritize emotional information. thats why old memories still vivid.
create emotional intensity during study.
emotion increases attention.
attention increases encoding.
encoding increases memory.
boredom created distraction and reduces attention and the distraction brain rarely memorize deeply.
so, the emotional ways are, attach personal meaning,
turn into story,
use humor,
create surprise or curiosity,
add intensity,
teach someone else,
visual, music, mood,
reward yourself,
link with past experiences,
make it dramatic,
make formula.
don't just think what should i learn, ask your self = how do i make the brain care?

5. the blurting method.
read, close the notes, recall everything, compare with the notes, repeat.
active recall, retrieval practice, error correction, encoding memory.

6. sleeping is our best friend.
when sleep the brain ..

7. the interchange.
4 hours study that, 4 hours study other, 4 hours study this - it's working because trains our brain to quick change.
